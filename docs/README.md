# MIDIWiki
?> [MIDIWiki](#) 是一个在线的维基知识库，包含关于 MIDI 的知识词条。所有的知识词条都是由所有参与过编辑 [MIDIWiki](#) 的作者们（包括本人）编制而成。您可以通过 [MIDIWiki](#) 来查阅文档信息。当然，您也可以参与其中，帮助我们编辑更多的知识词条，共同完善 [MIDIWiki](#) 维基知识库。

---
## 特性
- Ⅰ. 简单、方便、多平台
- Ⅱ. 支持移动端访问
- Ⅲ. 使用 Markdown 语言编写
- Ⅳ. 不定时更新维基知识库
- Ⅴ. 任何人都能编辑 MIDIWiki
